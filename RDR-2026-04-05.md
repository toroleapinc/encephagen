## 1. Summary

Day 5 was the most important day of the project. After 29 experiments, we reached a definitive negative result — and then found the path through it.

**The negative result:** With empirically validated dynamics (FC-FC=0.42 with real fMRI), the human connectome provides NO cognitive advantage over random wiring. 0/4 metrics significant in Experiment 29.

**The diagnosis:** Our regions are computationally identical. 200 random neurons per region = interchangeable noise generators. Structure can't help when all nodes are the same.

**The path forward:** Regional heterogeneity. The T1w/T2w myelination gradient gives each region a unique timescale. This is the ingredient that EVERY successful whole-brain model uses and we're missing.

---

## 2. Key Discovery: The Data Problem

We discovered that TVB96's connectivity weights are quantized to {1, 2, 3} — only 3 unique values. This carries almost no structural information.

Switched to neurolib's HCP-derived 80-region parcellation: 6,220 unique continuous weight values, 42 million x dynamic range. This immediately produced SC-FC validation (r=0.42 vs real fMRI) that we couldn't achieve with TVB96.

**Lesson:** The quality of the input data matters more than the sophistication of the model. 29 experiments on bad data produced 0 reliable findings. One test on good data produced validated dynamics.

## 3. Experiment 29: The Definitive Test

**Setup:** Neurolib 80 regions, continuous weights, FC-FC(emp)=0.42, 15 runs per condition, FDR corrected. Connectome vs degree-preserving random.

| Metric | Connectome | Random | p | d |
|--------|-----------|--------|---|---|
| Regional diff (CV) | 0.066 | 0.066 | 0.62 | -0.12 |
| Conditioning | +0.00002 | +0.00000 | 0.68 | +0.16 |
| Discrimination | 13.1% | 16.8% | 0.15 | -0.61 |
| Working memory | 73.5% | 4.0% | 0.19 | +0.57 |

0/4 FDR-significant. But working memory shows medium effect size (d=+0.57) favoring connectome — the one lead worth pursuing.

## 4. First-Principles Analysis

### Why structure doesn't help (currently)

The fundamental error: all regions are computationally identical. With 200 random neurons per region and identical parameters (tau_m=20ms everywhere), every region is a statistically interchangeable random network. The connectome routes signals between identical nodes — routing can't create differentiation when the nodes are the same.

The fly model works because **each neuron is unique.** The human brain works because **each region is unique** — different cell types, receptor densities, intrinsic timescales, layer composition.

We removed the uniqueness. That's why structure doesn't help.

### What the frontier shows

**BT-SNN (Zhao et al. 2024)** used Allen mouse connectome in a spiking RL agent and OUTPERFORMED random topologies. Key difference: each region has different computational properties.

**Consensus from Deco/Chaudhuri/Murray literature:** Regional parameter gradients (T1w/T2w → tau_m) are NECESSARY for structure to matter.

**No one has demonstrated cognitive advantage in a full spiking simulation with identical parameters.** We're not failing because our approach is wrong — we're failing because we're testing something that the literature already shows doesn't work.

### The fix

Implement the Murray/Chaudhuri timescale gradient. One parameter (tau_m) varies from 10ms (sensory cortex) to 30ms (prefrontal cortex) based on T1w/T2w myelination ratio from HCP data. This gives regions their unique computational identity. Then the connectome's routing matters because it routes between DIFFERENT processors, not identical ones.

## 5. What We're Exploring Is Genuinely Unknown

The expert panel and literature review confirm: **no one has built a functioning miniature human brain from macro-scale connectome data.** The closest attempts:
- Allen Institute: 10M neuron mouse simulation (Fugaku supercomputer) — dynamics only, no cognition
- BAAIWorm: 302-neuron worm — behavior emerges but required parameter fitting
- Eon Systems fly: 140K neurons embodied — hardwired sensorimotor, no learning
- BT-SNN: Connectome-topology spiking RL — works, but mesoscale abstraction

We are attempting something that has NOT been done. This isn't failure — it's exploration. The 29 experiments narrowed the search space: we now know what DOESN'T work (identical parameters, quantized weights, local-dominant connectivity) and what SHOULD work (regional heterogeneity, continuous weights, connectome-dominant architecture).

---

## 6. Revised Path (First Principles)

### Priority 1: Regional heterogeneity
The minimum change the literature says is necessary. Tau_m varies by region from T1w/T2w data. This transforms interchangeable regions into a heterogeneous computational network where routing matters.

### Priority 2: Test innate dynamics first, then learning
Stop testing learned tasks on an untrained brain. Test whether the connectome produces human-specific INNATE dynamics:
- Resting-state oscillations
- Sensory-to-frontal propagation hierarchy
- Default mode network patterns

### Priority 3: Learning as a scaffold test
The connectome's role may not be to compute better — it may be to LEARN better. Test learning speed: connectome vs random architecture trained on identical tasks. The 先天 × 后天 hypothesis predicts the connectome should learn faster, not perform better at birth.

## 8. What We've Learned Across 29 Experiments

### Robust findings:
1. SC-FC validation requires continuous weights (6K+ unique values, not 3)
2. FC-FC(empirical) = 0.42 achievable with neurolib80 + gc=10
3. Identical parameters guarantee structure irrelevance
4. dMRI tractography is sufficient for realistic FC but not for cognitive advantage

### Invalidated claims:
1. "Structure creates regional differentiation" — only on TVB96 with subcortical, disappears on neurolib80
2. "E-prop reveals conditioning advantage" — parameter-dependent (gc=0.15 only, unrealistic SC-FC)
3. "Structure IS intelligence" — not supported at any resolution we tested

### Genuine unknowns (what makes this research):
1. Does regional heterogeneity (tau_m gradient) unlock structural advantage?
2. Does the connectome provide a LEARNING scaffold (先天 × 后天)?
3. Can macro-scale structure produce human-specific innate dynamics?

These are real scientific questions that no one has answered.
